# Adding Big Numbers

> Write a function that returns the sum of two unusually large numbers.
> * The return type must be a string.
> * The input is a string of only digits.
> * The input numbers are positives.
> * The input numbers are *big*.
>
> Example
> ```
> add("123", "321"); -> "444"
> add("11", "99"); -> "110"
> ```
